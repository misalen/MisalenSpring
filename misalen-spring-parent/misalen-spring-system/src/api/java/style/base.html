接口文档基础说明

一.基本规则及配置

1. 基础地址:

测试:http://.......
正式:http://.......

2. 基础定义:

MD5加密为32位大写
文档中Date类型参数和返回值都为Long类型13位时间戳
用到的字典数据会在基础配置接口中返回

二.基本请求

1. 请求协议: 

http:// -> POST -> JSON

2. 请求格式

{
	"filter": {
		"imei": "11111111111",//设备IMEI
		"macCode": "Android",//设备类型 Android/iOS
		"macTime": 1459222265030,//设备时间
		"nonceStr":"1234567890",//十位随机字符串
		"sign":"05DD17D4F711CDC17C42B2483BE3FA5E"//认证串
	},
	"data":{
		...//接口文档中的请求数据
	}
}

sign计算方式:
sign =MD5(imei+macCode)+MD5(nonceStr+String(macTime)) 现在的sign是64位
sign =MD5(sign)

三.基本返回

1.基本结构(json数据)

{
    "errorCode": 200,//错误码 200为成功 小于1000位系统错误(404等) 大于等于1000位为接口错误
    "message": "ok",//错误信息
    "data": ...//接口文档中的返回数据
}
